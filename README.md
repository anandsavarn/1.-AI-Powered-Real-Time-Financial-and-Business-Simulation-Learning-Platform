## 📘 What Is Predictor.com?

**Predictor.com** is a patent-filed AI-powered platform that bridges the gap between financial theory and real-world market application.

Most trading simulators let you click "buy" and "sell" — but never explain **why** a trade worked or failed. Most finance courses teach theory — but never let you practice with live data.

**Predictor.com does both:**

- 🧠 Real-time stock market simulation connected to live NSE/BSE prices
- 🤖 AI that explains every trade decision in plain English
- 📊 Pre-action risk analysis before you commit
- 🔬 Post-action diagnostics that diagnose exactly what went wrong
- 💰 Virtual point economy — zero financial risk, 100% real market conditions

> 🏛️ **Filed as a complete patent at Lovely Professional University IPR Cell — provisional patent filing recommended.**

## 🤖 AI Engine — The Core Innovation

The AI engine is what makes Predictor.com fundamentally different from other simulators.

### Before Every Trade — Pre-Action Risk Analysis
```python
risk = pre_action_analysis(cost, user["points"])
# Returns:
# → Risk level (Low / Medium / High)
# → Market alignment score
# → Resource feasibility assessment
# → Historical success probability
```

### After Every Trade — Post-Action Diagnostics
```python
diagnosis = post_action_diagnostics(buy_price, current_price)
# Returns:
# → What led to profit or loss
# → Market volatility impact
# → Timing error analysis
# → Emotional decision detection (panic sell, FOMO buy)
# → Step-by-step corrective recommendations
```

### Self-Improving AI
The system anonymizes successful strategies from all users, finds common winning patterns, and uses them to help other users facing similar scenarios — creating a **collective intelligence knowledge base**.

## 🔑 Key Code Snippets

### User Registration — Virtual Economy Start
```python
@app.route("/register", methods=["POST"])
def register():
    username = request.json["username"]
    users[username] = {
        "points": 100000,      # Virtual investment capital
        "portfolio": {}
    }
    return jsonify({"message": "Registered", "points": 100000})
```

### Live Price Fetch
```python
@app.route("/price/<symbol>")
def price(symbol):
    price = get_live_price(symbol)
    return jsonify({"symbol": symbol, "price": price})
```

### Buy Simulation with AI Analysis
```python
@app.route("/buy", methods=["POST"])
def buy():
    data = request.json
    user = users[data["username"]]
    price = get_live_price(data["symbol"])
    cost = price * int(data["qty"])

    risk = pre_action_analysis(cost, user["points"])   # AI pre-check

    if user["points"] >= cost:
        user["points"] -= cost
        user["portfolio"][symbol] = user["portfolio"].get(symbol, 0) + qty
        diagnosis = post_action_diagnostics(price, price)  # AI diagnosis
        return jsonify({
            "status": "SUCCESS",
            "price": price,
            "points_left": user["points"],
            "risk_analysis": risk,       # AI explains the risk
            "diagnosis": diagnosis       # AI explains the outcome
        })
```

## 🚀 Installation & Setup

```bash
# Clone the repository
git clone https://github.com/Anandsavran/Predictor.com.git
cd Predictor.com

# Install backend dependencies
pip install -r backend/requirements.txt

# Run the Flask backend
cd backend
python app.py

# Open frontend
# Open frontend/index.html in your browser
# OR serve it with Live Server (VS Code)
```

**API Endpoints:**

| Method | Endpoint | Description |
|---|---|---|
| POST | `/register` | Register user, get 100,000 virtual points |
| GET | `/price/<symbol>` | Fetch live stock price |
| POST | `/buy` | Simulate buying a stock with AI analysis |
| POST | `/sell` | Simulate selling with diagnostics |
